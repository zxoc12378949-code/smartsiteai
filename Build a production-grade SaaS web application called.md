# Build Request

Build a production-grade SaaS web application called:

# SmartSite AI

Construction Safety Inspection & Defect Management Platform

The application should look and behave like a modern enterprise SaaS product used by construction companies, site managers, inspectors, and supervisors.

This is NOT a landing page.

This is NOT a presentation website.

This is NOT a portfolio.

Build a fully designed multi-page web application with realistic workflows, responsive layouts, modern dashboards, and professional UX.

---

# Product Vision

SmartSite AI is a cloud-based construction safety inspection platform.

The system digitizes site inspections, defect tracking, risk analysis, and safety reporting.

The platform helps construction teams:

- Conduct inspections
- Record defects
- Assign corrective actions
- Monitor safety risks
- Generate reports
- Track project safety performance

The design should feel similar to:

- Autodesk Construction Cloud
- Procore
- Monday.com
- Notion
- Jira
- Linear
- Asana
